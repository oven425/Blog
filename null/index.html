Develope Note

forwardRef useImperativeHandle

一般要使用html的tag是直接用useRef即可,如以下程式碼

const default App = ()=>{
    const btn = useRef(null)
    return(
        <div>
            <button ref={btn}>Button</button>
        </div>)
}

但通常會自己包一些component,例如以下

export const ChildWindow = ()=>{
    return (
        <div>
            ChildWindow
        </div>
    )
}

如果要對ChildWindow使用useRef,則必須要把ChildWindow改成下面這樣

export const ChildWindow = forwardRef(()=>{
    return (
        <div>
            ChildWindow
        </div>
    )
})

加上了forwardRef外部才能使用

const default App = ()=>{
    const cw = useRef(null)
    return(
        <div>
            <ChildWindow ref={cw}>Button</ChildWindow>
        </div>)
}
